
# SourceCompare.vb — A .Net 10 Utility for visually comparing two text files.



## Why this exists

I just started moving my projects from VS 2019 to VS 2026.  For over a decade, VS supported the old
but still useful Visual Source Safe source controller, which provided a built-in source-file compare
utility.  Since VS 2022, Microsoft has moved to Git instead, which only offers a version in Git Changes,
but not a general-purpose utility, so I wrote my own.  Then, because I wanted a version for the .Net 10
framework, I duplicated it under that platform.  This is that version.
  
## What This Is

- A `Visual Studio` project
- Requires PromptedTextBox Net 10.dll, available from another of my repositories.
